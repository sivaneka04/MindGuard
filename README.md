# MindGuard

MindGuard is an AI-Driven Digital Wellness Desktop Application developed as a Final Year Project.

## Features

- Screen Time Monitoring
- Facial Stress Detection
- AI Chatbot Support
- Break Reminder
- Wellness Report Generation

## Technologies Used

- Python
- CustomTkinter
- OpenCV
- Face++ API
- SQLite
